<h1>AE1-UD4</h1>
<h2>Descripción</h2>
La clase CalculoMedia nos ofrece una solución para un análisis completo a un conjunto de datos de enteros.
<p>Operaciones que se realizan en el código: 
 Realiza las siguientes operaciones:
 <ul>
 <li>Identifica el valor máximo dentro de un array y determina si este se encuentra duplicado.</li>
 <li>Calcula la media de los elementos del arreglo.</li>
 <li>Evalúa si la media alcanza el aprobado (5.0) y se informa al ususario.</li>
 </ul>
</p>
<h2>Ejecución</h2> 
Al tener todos los datos definidos dentro del código y no necesitar ningún valor proporcionado por el usuario externo, podremos ejecutar directamente el código.

<p>En la refactorización del código se han observado repeticiones innecesarias de algunos procedimiento. Uso de variables extra que no aportaban valor significativo al código, por lo que se ha decidido comentar el código sobrante de nuestro programa.</p>
